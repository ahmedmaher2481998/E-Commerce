# ComfySloth

> ComfySloth is an E-commerce Website for selling furniture,powered by React,and
> consumes json data from a Api endpoint,the app has sing in/out with auth0, persist cart data using localStorage browser api,and use react Context with redux pattern to manage the app state .

## installation

this application was bootstrapped by CRA .

to install dependencies run

> npm install

to run the app in **development** mode ,run.

> npm start
